# pizza-sales-sql-analysis
An end-to-end SQL project analyzing pizza sales data to uncover business insights, including revenue analysis, sales trends, customer ordering patterns, and product performance using PostgreSQL.

## Project Overview

This project analyzes pizza sales data using SQL to derive meaningful business insights. The analysis focuses on revenue generation, sales trends, customer ordering behavior, and product performance.

## Objectives

* Analyze total orders and revenue generated.
* Identify best-selling and highest-priced pizzas.
* Examine order distribution by time and date.
* Determine category-wise and size-wise sales performance.
* Analyze cumulative revenue trends.
* Identify top-performing pizza types based on revenue.

## Tools Used

* PostgreSQL
* pgAdmin
* SQL

## Key Insights

* Total revenue generated from pizza sales.
* Most ordered pizza types and sizes.
* Peak ordering hours and daily sales trends.
* Revenue contribution by pizza category.
* Top-performing pizzas based on revenue.

## SQL Concepts Used

* Joins
* Aggregate Functions
* Common Table Expressions (CTEs)
* Window Functions
* Group By and Order By
* Subqueries

## Dataset

The dataset contains information about orders, order details, pizzas, and pizza categories.
